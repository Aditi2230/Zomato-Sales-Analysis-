# 🍽️ Zomato Sales Analysis Dashboard

## 🚀 Project Overview

This project focuses on analyzing Zomato's sales data to uncover key business insights related to customer behavior, restaurant performance, and delivery efficiency.

An interactive dashboard was built to help stakeholders make data-driven decisions.

---

## 📊 Key Features

* 📈 Sales trend analysis (daily/monthly)
* 🍕 Top-performing restaurants & food categories
* 👥 Customer segmentation (New vs Repeat)
* 🚚 Delivery performance analysis
* 💳 Payment mode distribution
* ⭐ Customer rating insights

---

## 🛠️ Tools & Technologies

* Microsoft Excel 
* Pivot Tables & Data Modeling
* Data Cleaning & Transformation

## 📈 Key Insights

* Top restaurants contribute a major share of total revenue
* Repeat customers generate consistent revenue streams
* Faster delivery times correlate with higher customer ratings
* Peak order times observed during evening slots

---

## 📂 Dataset

* Contains order-level data including:

  * Order ID, Date, City
  * Restaurant Name, Category
  * Order Amount, Delivery Time
  * Customer Rating, Payment Mode
